# Injective
- iff $\forall$x $\epsilon$ A, $\forall$y $\epsilon$ A (f(x) = f(y) $\rightarrow$ x = y)

# Surjective
- iff $\forall$z $\epsilon$ B, $\exists$x $\epsilon$ A (f(x) = z)

# Bijective
- iff f is both [[Functions#Injective]] and [[Functions#Surjective]]
